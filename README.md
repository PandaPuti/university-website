# Eduford University - Multi-Page Responsive Website 🏫

A comprehensive, fully responsive university website featuring dynamic navigation, interactive forms, and modern UI/UX principles.

## 🚀 Core Features
* **Multi-Page Navigation:** Seamless transitions between Home, About, Course, Blog, and Contact pages.
* **Asynchronous Form Handling:** The Contact page uses the JavaScript `Fetch API` to simulate real-time data submission with loading states.
* **Responsive Design:** Mobile-first approach using Flexbox and Media Queries for cross-device compatibility.
* **Interactive UI:** Custom hover effects, Font Awesome integration, and a JavaScript-powered side-toggle mobile menu.

## 🛠️ Technical Implementation
* **Layout:** Advanced CSS Flexbox (nesting rows and columns).
* **Styling:** Linear gradients, Google Fonts, and custom transition animations.
* **JavaScript:** DOM manipulation for menu toggling and Fetch/Promises for form processing.

## 🧠 Major Milestones
* Managed global styles across multiple HTML files.
* Implemented a "Loading State" pattern for better user feedback.
* Successfully debugged complex nesting issues caused by unclosed HTML tags.
